 * stream_mssg_on
 *
 * Start printing a "debugging" message.
 *
  stream_mssg_on 
 * stream_mssg_off
 *
 * Stop printing a "debugging" message.
 *
  stream_mssg_off 
 * z_output_stream, open or close an output stream.
 *
 *	zargs[0] = stream to open (positive) or close (negative)
 *	zargs[1] = address to redirect output to (stream 3 only)
 *	zargs[2] = width of redirected output (stream 3 only, optional)
 *
  z_output_stream 
 * stream_char
 *
 * Send a single character to the output stream.
 *
  stream_char 
 * stream_word
 *
 * Send a string of characters to the output streams.
 *
  stream_word 
 * stream_new_line
 *
 * Send a newline to the output streams.
 *
  stream_new_line 
 * z_input_stream, select an input stream.
 *
 *	zargs[0] = input stream to be selected
 *
  z_input_stream 
 * stream_read_key
 *
 * Read a single keystroke from the current input stream.
 *
  Read key from current input stream  Verify mouse clicks  Copy key to the command file  Handle timeouts  Handle hot keys  Return key  stream_read_key 
 * stream_read_input
 *
 * Read a line of input from the current input stream.
 *
  Remove initial input from the transscript file or from the screen  Read input line from current input stream  Verify mouse clicks  Copy input line to the command file  Handle timeouts  Handle hot keys  Copy input line to transscript file or to the screen  Return terminating key  stream_read_input  stream.c - IO stream implementation
