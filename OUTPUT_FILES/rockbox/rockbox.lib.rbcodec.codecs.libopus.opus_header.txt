 Header contents:
  - "OpusHead" (64 bits)
  - version number (8 bits)
  - Channels C (8 bits)
  - Pre-skip (16 bits)
  - Sampling rate (32 bits)
  - Gain in dB (16 bits, S7.8)
  - Mapping (8 bits, 0=single stream (mono/stereo) 1=Vorbis mapping,
             2..254: reserved, 255: multistream with no mapping)

  - if (mapping != 0)
     - N = totel number of streams (8 bits)
     - M = number of paired streams (8 bits)
     - C times channel origin
          - if (C<2*M)
             - stream = byte/2
             - if (byte&0x1 == 0)
                 - left
               else
                 - right
          - else
             - stream = byte-M
 Only major version 0 supported.  Multi-stream support For version 0/1 we know there won't be any more data
     so reject any that have data past the end. Version is 1  Multi-stream support  This is just here because it's a convenient file linked by both opusenc and
   opusdec (to guarantee this maps stays in sync).  1.0 mono    2.0 stereo  3.0 channel ('wide') stereo  4.0 discrete quadraphonic  5.0 surround  5.1 surround  6.1 surround  7.1 surround (classic theater 8-track)  Copyright (C)2012 Xiph.Org Foundation
