 This is a TIFF image file loading framework  LOAD_TIF_DYNAMIC 
 * These are the thunking routine to use the SDL_RWops* routines from
 * libtiff's internals.

	 * We don't want libtiff closing our SDL_RWops*, but if it's not given
         * a routine to try, and if the image isn't a TIFF, it'll segfault.
	  The error message has been set in SDL_RWFromFile  turn off memory mapped access with the m flag  Retrieve the dimensions of the image from the TIFF tags  libtiff loads the image upside-down, flip it back  See if an image is contained in a data source  Load a TIFF type image from an SDL datasource  LOAD_TIF  !defined(__APPLE__) || defined(SDL_IMAGE_USE_COMMON_BACKEND)
